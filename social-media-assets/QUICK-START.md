## üì± 15-Minute Quote Card Workflow

### Tools Needed
- Canva (free)

### Steps

1. **Open Canva** ‚Üí Create Design ‚Üí 1080 x 1080

2. **Choose Template** from `templates/canva-templates.md`

3. **Copy Quote Text** from `chapter-XX/quote-cards.md`

4. **Apply Brand**:
   - Background: Dark gradient or solid `#1E293B`
   - Text: White
   - Accent: `#3B82F6` (blue)

5. **Export**: PNG, High Quality

6. **Copy Caption** from same file

7. **Post!**

## üìä 45-Minute Carousel Workflow

### Tools Needed
- Canva (free)

### Steps

1. **Open Canva** ‚Üí Create Design ‚Üí 1080 x 1080

2. **Create Cover Slide**:
   - Bold title
   - Hook question
   - "Swipe ‚Üí" indicator
   - Progress dots

3. **For Each Content Slide**:
   - Copy from `chapter-XX/linkedin-carousel.md`
   - One main point per slide
   - Slide number (2/8, 3/8, etc.)
   - Consistent styling

4. **Create CTA Slide**:
   - Follow prompt
   - Save reminder
   - Question for engagement

5. **Export All**: Download ‚Üí All Pages ‚Üí PNG

6. **Copy Caption** from carousel file

7. **Upload to LinkedIn/Instagram**

### Pro Tips
- Use Canva's duplicate feature for consistency
- Create 1 template, reuse for all chapters
- Carousel = highest engagement on LinkedIn

## üê¶ 30-Minute Twitter Thread Workflow

### Tools Needed
- Twitter/X
- Excalidraw or Canva (for header image)

### Steps

1. **Create Header Image** (optional but recommended):
   - Size: 1200 x 675
   - Use diagram from `chapter-XX/diagrams.md`
   - Or simple text card in Canva

2. **Copy Thread** from `chapter-XX/twitter-thread.md`

3. **Post Tweet 1** with image

4. **Reply with remaining tweets** (Thread format)

5. **Add relevant hashtags** to last tweet only

### Pro Tips
- Schedule with Twitter's built-in scheduler
- First tweet = most important (hook + image)
- Space out threads throughout the week

## üñºÔ∏è 30-Minute Diagram Workflow

### Tools Needed
- Excalidraw (free, [excalidraw.com](https://excalidraw.com))

### Steps

1. **Open Excalidraw**

2. **Copy Specification** from `chapter-XX/diagrams.md`

3. **Build Diagram**:

   - Follow color guide from `templates/excalidraw-style-guide.md`
   - Keep it simple and readable

4. **Export**:
   - Select All (Ctrl+A)
   - Export Image
   - Scale: 2x
   - Format: PNG
   - Include Background: Yes

5. **Post** with simple caption:
   - Brief explanation
   - "Save this! üîñ"
   - Relevant hashtags

## üîÑ Batch Creation Day (4 Hours)

Create a full week of content in one session:

### Hour 1: Quote Cards (4 cards)
1. Open Canva
2. Create 1 template
3. Duplicate 3 times
4. Fill with different quotes
5. Export all

### Hour 2: Carousels (1 carousel)
1. Open Canva
2. Create 10 slides
3. Follow carousel template
4. Export all pages

### Hour 3: Twitter Threads (2 threads)
1. Draft in notes app
2. Create header images
3. Save as drafts in Twitter

### Hour 4: Diagrams (3 diagrams)
1. Open Excalidraw
2. Create 3 simple diagrams
3. Export as PNG

**Result**: 7 days of content ready to post!
