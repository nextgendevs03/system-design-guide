|-------|-----|---------|

- Rounded corners preferred

## ğŸ”§ Tools Setup

### Excalidraw (Architecture Diagrams)
1. Go to [excalidraw.com](https://excalidraw.com)
2. Import our library: `templates/excalidraw-library.excalidrawlib` (coming soon)
3. Use hand-drawn style (default)
4. Export as PNG (2x scale for quality)

### Canva (Social Cards)
1. Use templates from `templates/canva-templates.md`
2. Stick to brand colors
3. Export as PNG

### Mermaid to Image
1. Go to [mermaid.live](https://mermaid.live)
2. Paste Mermaid code from chapters
3. Download as SVG/PNG
